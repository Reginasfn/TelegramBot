# 🎬 Telegram PopcornBot 🍿

## 📌 Описание проекта

**Telegram PopcornBot** — это Telegram-бот для рекомендаций фильмов.  
Бот помогает пользователю находить интересные фильмы с помощью различных команд.

Функционал бота включает:

- получение случайного фильма;
- просмотр фильма дня;
- подбор фильма по параметрам;
- угадывание фильма по описанию пользователя;

Бот использует внешнее API базы данных фильмов и модель искусственного интеллекта для обработки текстовых запросов.

## ⚙️ Функционал бота

Бот поддерживает следующие команды:

| Команда | Описание |
|--------|---------|
| `/start` | запуск бота и отображение главного меню |
| `/random` | случайный фильм |
| `/day` | фильм дня |
| `/params` | подбор фильма по параметрам |
| `/guess` | угадай фильм по описанию |
| `/help` | справка по командам |

## 🎬 Используемое API

### TMDB API (The Movie Database)

В проекте используется API сервиса **TMDB — The Movie Database**.

TMDB — это одна из крупнейших открытых баз данных фильмов и сериалов.

API позволяет получать:

- информацию о фильмах
- рейтинги
- жанры
- описания
- постеры
- рекомендации

В проекте API используется для:

- получения случайных фильмов
- выбора фильма дня
- поиска фильмов по параметрам
- отображения информации о фильме

Документация API:  
https://developer.themoviedb.org/

---

## 🧠 Используемая модель трансформера

В проекте используется трансформерная модель:

**BERT (Bidirectional Encoder Representations from Transformers)**

### Задача модели

Модель применяется для **анализа и классификации текста**.

Она используется в функции `/guess`, где пользователь вводит описание фильма, а бот пытается определить, о каком фильме идет речь.

Пример:

Пользователь пишет:

> Фильм про космос где астронавт остался один на Марсе

Бот анализирует текст и пытается определить соответствующий фильм.
